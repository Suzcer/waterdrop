## 第 5 章 索引

索引是用于提升查询效率的一种存储结构。

### 索引简介

在 MongoDB 中，不使用索引的查询称为**集合扫描**，这意味要扫描所有数据。

创建索引

```json
db.users.createIndex({"username" : 1})
```

创建复合索引

如果查询中有多个排序方向或者查询条件中有多个键，那么复合索引会非常有用。

```json
db.users.createIndex({"age" : 1, "username" : 1})
```
